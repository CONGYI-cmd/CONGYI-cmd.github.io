Project Concept/Motivation:

Trash can robot in Disneyland

I still remember when I first went to Disneyland, there was a trash can robot which can talk,
move, and open & close automatically. I always want to have something like that at home.
Therefore, for this project, I would like to design and implement one by myself. My trash can robot
will use an arduino and several electronic components to support its functionalities. When
it knows someone is nearby, the robot would be activated and start “talking” with the person.

Here is a video shows how the trash can robot interacts with people in Disneyland:
https://www.youtube.com/watch?v=dZnDtZYWIfc

If time allowed: add LED strips and use the strips to show the robot’s emotion. The LED will change color and flash corresponding to the content on the LCD.

Anticipated Bill of Materials:

Arduino (for running the firmware)
Breadboard (for implementing the circuit)
Ultrasonic Sensor (for sensing the object in front of the robot)
Servo Motor (to open & close the lid automatically)
LCD (for showing texts to users)
LED Strips (if I have time, for showing the emotion of the robot with color)

Timeline:

By March 1, prepare all the materials, finish the prototype design
March 1 to March 5, build the circuit and implement the code
March 5 to March 8, debug and optimize the product. (Add the LED strip to the robot if time allowed)

Contingency Plans:

If the Ultrasonic Sensor is not able to sense the object in the front, I will use a motion sensor.
If the plastic trash can is too difficult to add electric components on it, I will use a paper box.
